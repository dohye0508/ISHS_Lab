# 알고리즘 문제 해결 및 코드 작성 규칙

모든 알고리즘 답변은 다음 형식을 엄격히 준수하여 작성해야 함.

## 1. 문서 상단 구성
- [문제 제목]: 문제의 핵심 제목.
- 문제 설명: 해결을 위한 핵심 아이디어 설명.
- 시간 복잡도: O(N) 형태의 빅오 표기법 명시.

## 2. 입출력 예시
- [입력 예시] / [출력 예시] 형태로 작성.

## 3. 코드 구현 (Python)
- 구조 간소화: `if __name__ == "__main__":`이나 `solve()` 함수 등으로 감싸지 않고 바로 코드를 작성한다.
- 입출력 효율화: `import sys`와 `input = sys.stdin.readline`은 포함한다.
- 주석: 아래의 필수 주석 섹션만 포함한다.
    1. # 데이터 입력
    2. # 문제 해결 로직
    3. # 결과 출력
- 코드 예시:
    import sys
    input = sys.stdin.readline

    # 데이터 입력
    n = int(input())
    
    # 문제 해결 로직
    result = ...
    
    # 결과 출력
    print(result)
